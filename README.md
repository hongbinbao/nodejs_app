nodejs demo
==========

demo of node.js

#### Prerequisites (Unix only):

    * GCC 4.2 or newer
    * Python 2.6 or 2.7
    * GNU Make 3.81 or newer
    * libexecinfo (FreeBSD and OpenBSD only)
    
#### Installation
    1: download current version from http://nodejs.org/
    2: $ ./configure
       $ make
       $ make install

#### Launch demo
    1: cd to "nodejs_demo/"
    2: start server
       $ node index.js
    2: open browser client and type
       http://localhost:8889/
       http://localhost:8889/start
